##  Weather Forecast with Trip Planner

A modern Weather Forecast and Trip Planner application built using React, TypeScript, Vite, Tailwind CSS, and the OpenWeather API.

The application provides real-time weather information, hourly forecasts, weekly forecasts, and helps users plan trips based on weather conditions.

# Features

-  Search weather by city
-  Current temperature
-  Weather conditions
-  Humidity
-  Wind speed
-  7-Day Forecast
-  Hourly Forecast
-  Trip Planning
-  Responsive Design
-  Fast Vite Development Server

# Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- OpenWeather API

##  Installation

Clone the repository

-bash
git clone https://github.com/deepprakashds5/weather-forecast-trip-planner.git
-

Move inside the project

-bash
cd weather-forecast-trip-planner
-

Install dependencies

-bash
npm install
-

Create a '.env' file

-env
VITE_OPENWEATHER_API_KEY=YOUR_API_KEY
-

Start the development server

-bash
npm run dev
-
