## Weather Forecast with Trip Planner

A responsive React + TypeScript application that provides real-time weather forecasts and recommends destinations for trip planning.

# Features

 Current weather
 Hourly forecast
 7-day forecast
 Weather details
 Trip planner
 Destination recommendations
 Responsive UI

# Tech Stack

 React
 TypeScript
 Vite
 Tailwind CSS

# Installation

-bash
npm install
npm run dev
